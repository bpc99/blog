发表于 2020-09-06 分类于 web 本文字数 959 阅读时长 ≈ 3分钟

Es6 不仅添加一些常用的方法、对象，还对 JavaScript 的基本变量进行了扩展，在原先 var 的基础上又提出了 let 和 const 关键字。那么这两个关键字到底有什么作用，到底解决了什么问题呢？
