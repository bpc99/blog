Hexo一键部署

发表于 2021-01-27 分类于 Linux 本文字数 1.2k 阅读时长 ≈ 4分钟

使用 Hexo 搭建完成博客，平常会更新一些文章，但是其部署却十分的麻烦，由于 Hexo 是将 Mackdown 语法打包为 HTML 界面，每个文章对应着一个界面，这样部署起来很不方便，基本每次修改文章，都要重新登陆服务器覆盖，既然那么多重复步骤，那么 Hexo 可不可以一行命令帮我们将博客部署到服务器上呢？这是肯定的，Hexo 也给出了相应的配置。
