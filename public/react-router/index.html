发表于 2021-01-29 分类于 web 本文字数 1.4k 阅读时长 ≈ 5分钟

导航守卫在日常开发中经常会被使用到，比如一个界面需要登陆之后或需要某种权限才能进行访问，这样我们可以很简单的使用导航守卫实现。

导航守卫

vue 的导航守卫主要分为 3 种，分别为：全局导航守卫、路由导航守卫、组件导航守卫，而最为常用的便是全局导航守卫中的beforeEach，每次界面的跳转都会经过里面的逻辑，只有通过才会进行跳转。
